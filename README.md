# Chromosome-instability-correlation-analysis

The repository contains R bioinformatic analysis for the paper by M. Liskovykh et al. published in PNAS Nexus:

Mikhail Liskovykh, Natalia Y Kochanova, Chih-Yuan Chiang, Anjali Dhall, Vasilisa Aksenova, Yu-Chi Chen, William C Reinhold, Mary Dasso, Anish Thomas, Ken Chih-Chien Cheng, Yves Pommier, William C Earnshaw, Vladimir Larionov, Natalay Kouprina, A genome-wide RNAi screen for novel CIN genes using human artificial chromosome, PNAS Nexus, Volume 4, Issue 12, December 2025, pgaf345, https://doi.org/10.1093/pnasnexus/pgaf345

The folder contains the code used in computing the correlation analysis (fig. 4 and fig. S3).
